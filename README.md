# Generador de Números Pseudoaleatorios

Un proyecto de simulación construido con Vite, React, Tailwind CSS 4 y Recharts.

## Características
* **Generadores**: Congruencial Lineal, Congruencial Multiplicativo, Cuadrados Medios.
* **Pruebas de Uniformidad/Independencia**: Chi-Cuadrado, Kolmogorov-Smirnov, Póker.
* **Componentes Visuales**: Gráficos de dispersión y tablas interactivas.

## Ejecución
1. Instalar dependencias: `npm install`
2. Correr modo desarrollo: `npm run dev`
